## FAQ (Use in Marketing)

**Q: How is it 31% faster than Copilot?**
A: We run locally on Ollama (no network latency) and use smaller, smarter models trained for code analysis. Copilot adds 12+ ms of network overhead alone.

**Q: If it's so good, why isn't everyone using it?**
A: It's brand new! Copilot has brand recognition and GitHub integration. Lightning is open-source and builds on similar principles, just more efficiently.

**Q: Does the 4% accuracy gap matter?**
A: For most use cases, no. You gain 31% speed and 62x cost savings. The quality difference is within acceptable range for code suggestions that you review anyway.

**Q: What about privacy?**
A: Zero data leaves your machine. All analysis happens locally. No telemetry. This is a major differentiator.

**Q: Can I use it at work?**
A: Yes! MIT license means commercial use is fine. Many companies are already exploring open-source alternatives to Copilot for this reason.

**Q: Does it need internet?**
A: No. Once installed, Lightning runs completely offline (after downloading Ollama models once).
