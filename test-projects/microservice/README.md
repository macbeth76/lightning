# gRPC Microservice

**Ground Truth**: gRPC microservice with Kubernetes deployment

## Architecture
- Service: service.ts (gRPC handlers)
- Protocol: service.proto (gRPC definition)
- Deployment: Kubernetes

## Dependencies
- @grpc/grpc-js (gRPC framework)
- @grpc/proto-loader (protocol loader)
- protobufjs (protocol buffers)

## Project Type
Distributed microservice with gRPC protocol
